## Ionic 4 BMI Calculator App

### About this repo
This repo is the project code of an Ionic 4 app that retrieves and shows the up-to-date price rate of Bitcoin to USD, AUD, GBP, EUR, and TRY.

### Installation
```
npm install
```

### Browse app
```
ionic serve
```
or

```
ionic lab
```
